# CaprioLibras

Projeto de integração com VLibras para tornar conteúdo web acessível à comunidade surda.

## Descrição

Este projeto contém um dicionário de Libras e exemplos de integração com o VLibras, uma ferramenta que traduz conteúdo de português para Libras automaticamente.

## Arquivos

- `libras-dictionary.js` - Dicionário com termos em português e suas correspondências em Libras
- `pagina-vlibras.html` - Página de exemplo com VLibras integrado

## Como usar

1. Inclua o script do VLibras na sua página HTML
2. Configure o widget com as opções desejadas
3. Use o dicionário para mapear termos específicos
