Esta é a página principal do projeto CaprioLibras, desenvolvido para tornar conteúdo web acessível à comunidade surda através da integração com VLibras.

Como usar:

Clique no botão azul que aparece no canto da tela para abrir o intérprete de Libras.

Você pode testar traduzindo os textos desta página.

Sobre o projeto

O CaprioLibras é uma iniciativa do Colégio Hercília Franca para promover a inclusão digital e tornar o conteúdo educacional acessível para todos os alunos, incluindo a comunidade surda.

Recursos

  • Dicionário de termos em Libras
  • Integração com VLibras
  • Páginas web acessíveis
  • 📄 Visualizador de Documentos
  • 🤖 Visualizador Automático

Obrigado por visitar! Este projeto está disponível no GitHub para contribuições da comunidade.
